**1.** O que é o Node.js?
é um ambiente de execução (runtime) JavaScript de código aberto e multiplataforma que permite executar códigos JavaScript no lado do servidor (backend), fora do navegador

**2.** O Node.js usa qual linguagem de programação?
 JavaScript 
 
**3.** O Node.js roda no navegador ou fora dele?
Ele é um ambiente de execução (runtime) que permite que o JavaScript seja executado diretamente no sistema operacional (servidor, computador desktop, etc.), sem precisar de um navegador web (como Chrome ou Firefox) para 
funcionar. 

**4.** Para que serve o Node.js?
O Node.js serve primordialmente como um ambiente de execução (runtime) para rodar código JavaScript no servidor (backend)

**5.** Qual comando é usado para executar um arquivo Node.js no terminal?
node nome_do_arquivo.js

**6.** O que é o **npm**?
é o gerenciador de pacotes padrão para o ambiente de execução Node.js e o maior registro de software de código aberto do mundo

**7.** O que significa a palavra **módulo** no Node.js?
No Node.js, um módulo é uma unidade de código funcional, encapsulada e reutilizável, geralmente contida em um arquivo JavaScript separado. Eles permitem organizar grandes aplicações em arquivos menores, mais organizados e gerenciáveis. 

**8.** O Node.js é mais usado no **front-end** ou no **back-end**?
back-end

**9.** O que é o arquivo **package.json**?
 é o coração de qualquer projeto Node.js e de muitos projetos de frontend modernos.
 
**10.** O Node.js é gratuito ou pago?
totalmente gratuito
